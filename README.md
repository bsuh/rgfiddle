rgfiddle
========
IDE for robotgame.org

Installation
------
    cd ~
    git clone https://github.com/bsuh/rgfiddle.git
    cd rgfiddle
    pip install -r requirements.txt
    git submodule init
    git submodule update
    python backend.py

Then you can visit `http://localhost:8080/`

Keeping rgkit up to date
------
    cd ~/rgfiddle
    git submodule update
